# 约束基线

> **基线约束是项目级别的硬性规则**，所有业务单元和代码都应遵循这些约定。
> 请根据你的项目实际情况填写约束内容。这里只提供模板骨架和方向指引。

## 约束分类

| 约束 | 优先级 | 说明 |
|------|--------|------|
| 技术约束 | ⭐⭐⭐ | 编码规范、架构决策、技术栈选择 |
| 部署约束 | ⭐⭐⭐ | 环境配置、CI/CD、发布流程 |
| 数据库约束 | ⭐⭐ | 表命名、迁移策略、ER 规范 |
| 产品约束 | ⭐⭐ | 产品定位、用户画像、边界定义 |

> 提示：约束基线因人而异、因项目而异，每个约束提供 **约束内容 + 原因 + 适用范围** 即可。
