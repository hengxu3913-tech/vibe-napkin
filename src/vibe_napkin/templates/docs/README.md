# {{ project_name }} — 文档中心

> 本项目使用 vibe-napkin 🧻 管理技术文档和知识库。

## 📂 文档结构

| 目录 | 说明 |
|------|------|
| `业务单元/` | 每个功能点的纵向贯穿文档（产品+技术+部署+数据库） |
| `约束基线/` | 项目级别的硬性约束规范 |

## 使用指引

1. **开发功能 → 验收 → `vibe-napkin wipe`**：同步业务单元
2. **`vibe-napkin tidy`**：更新向量知识库
3. **`vibe-napkin save`**：存档当前进度
4. **`vibe-napkin load`**：读档继续开发
